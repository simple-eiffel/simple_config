simple_config

Foundation LayerVoid-Safe

Easy JSON-based configuration file management for Eiffel applications.

Features

  • Simple key-value configuration storage
  • JSON file format for human-readable configs
  • Type-safe getters for STRING, INTEGER, BOOLEAN, REAL
  • Default value support for missing keys
  • Nested sections for organized configurations
  • Load/save to files
  • Modification tracking

Installation

Add to your ECF file:

<library name="simple_config" location="$SIMPLE_CONFIG\simple_config.ecf"/>

Quick Start

class MY_APP
create
    make

feature
    make
        local
            cfg: SIMPLE_CONFIG
        do
            -- Load from file
            create cfg.make_with_file ("config.json")

            -- Read values with defaults
            print ("Server: " + cfg.string_value_or_default ("server", "localhost"))
            print ("Port: " + cfg.integer_value_or_default ("port", 8080).out)
            print ("Debug: " + cfg.boolean_value_or_default ("debug", False).out)

            -- Update values
            cfg.set_string ("server", "192.168.1.100")
            cfg.set_integer ("port", 9000)
            cfg.set_boolean ("debug", True)

            -- Save changes
            cfg.save
        end
end

API Reference

Creation

FeatureDescription
makeCreate empty configuration
make_with_file (path)Create and load from file

String Access

FeatureDescription
string_value (key): detachable STRINGGet string value
string_value_or_default (key, default): STRINGGet with default
set_string (key, value)Set string value

Integer Access

FeatureDescription
integer_value (key): INTEGERGet integer value
integer_value_or_default (key, default): INTEGERGet with default
set_integer (key, value)Set integer value

Boolean Access

FeatureDescription
boolean_value (key): BOOLEANGet boolean value
boolean_value_or_default (key, default): BOOLEANGet with default
set_boolean (key, value)Set boolean value

Real Access

FeatureDescription
real_value (key): DOUBLEGet real value
real_value_or_default (key, default): DOUBLEGet with default
set_real (key, value)Set real value

Sections

FeatureDescription
section (key): detachable SIMPLE_CONFIGGet nested section
set_section (key, config)Set nested section

File Operations

FeatureDescription
loadLoad from file_path
saveSave to file_path
save_to (path)Save to specific path

Status

FeatureDescription
has_key (key): BOOLEANDoes key exist?
is_modified: BOOLEANHas config been modified?
remove (key)Remove a key

Nested Sections Example

-- Create nested configuration
cfg: SIMPLE_CONFIG
db: SIMPLE_CONFIG

create cfg.make
create db.make
db.set_string ("host", "localhost")
db.set_integer ("port", 5432)
db.set_string ("name", "myapp")
cfg.set_section ("database", db)
cfg.save_to ("config.json")

-- Result (config.json):
-- {
--   "database": {
--     "host": "localhost",
--     "port": 5432,
--     "name": "myapp"
--   }
-- }
